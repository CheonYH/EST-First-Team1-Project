## 앱 설명
기존의 메모앱의 단점인 text와 title로만 검색을 한다는것이였습니다.
나중에 내가 메모 했던 것을 찾으려고 할때는 모든 text를 직접 읽으며 찾아야 했습니다.
저희는 이 문제점을 보완해보자는 아이디어에서 출발했습니다.
